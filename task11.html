Task 11 - Dynamic Link Libraries (DLLs)

This task involves creating a Dynamic Link Library (DLL) and using it in a different application. Below are the steps for creating the library and the app to call its methods.

üßæ Code: `Mymathlib.cs` (Library)

using System;

namespace mymathlib
{
    public class method1
    {
        public static long Add(long x, long y)
        {
            return (x + y);
        }

        public static long Mul(long x, long y)
        {
            return (x * y);
        }
    }
}

üßæ Code: `MyMathApp.cs` (Application)

using System;
using mymathlib;

class MyMathApp
{
    public static void Main(string[] args)
    {
        Console.WriteLine("Calling Method from mymathlib");

        if (args.Length != 2)
        {
            Console.WriteLine("Usage : MyMathApp  ");
            return;
        }

        long num1 = long.Parse(args[0]);
        long num2 = long.Parse(args[1]);

        long add = method1.Add(num1, num2);
        long mul = method1.Mul(num1, num2);

        Console.WriteLine("{0} + {1} = {2}", num1, num2, add);
        Console.WriteLine("{0} * {1} = {2}", num1, num2, mul);
    }
}
Note: In this example, the class method1 contains two static methods: Add and Mul, which are used in the MyMathApp application. The app takes two numbers as arguments and uses the DLL to perform the addition and multiplication.

  1. Open terminal and create a new WinForms project: dotnet new winforms -o first.
  2. Add the files: Mymathlib.cs (library) and MyMathApp.cs (application).
  3. To build the DLL, use the command: dotnet build.
  4. Run the app using the command: dotnet run --project MyMathApp.
